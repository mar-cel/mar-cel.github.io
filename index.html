About

Portfolio

Spot

Spot is a one-tap car location saving Android application. Users can also add notes and a picture to help for later. When it comes to finding the car, a route is drawn on the map from the user's current location. If needed, the user can open Google Navigation or look at the Google Street View of the car's location.

Tern

Tern is a travel application for planning places that a user may want to go to in the future. Places can be organized into collections, like songs in a playlist. Also, the user can view any collection or even all their places with markers on a map for a better visual representation. Tern is soon to be available on the Google Play Store (Android) and the App Store (iOS).

REST Roulette

A REST API server for playing roulette. Experiment with different bet types and wager amounts. Try it out at www.roulette.rip.
